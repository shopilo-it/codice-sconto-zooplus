shopilo-it / codice-sconto-zooplus

Script Python open-source per il recupero automatico di codici sconto Zooplus da shopilo.it. Restituisce coupon attivi in formato JSON.

Python 3.8+ MIT License requests + beautifulsoup4 shopilo.it
🔗 Dati live Zooplus su shopilo.it

Installazione e utilizzo rapido

# Installa le dipendenze
pip install requests beautifulsoup4

# Clona il repository
git clone https://github.com/shopilo-it/codice-sconto-zooplus
cd codice-sconto-zooplus

# Esegui lo script
python fetch.py

Oppure installa come modulo npm:

npm install codice-sconto-zooplus

# Utilizza in Node.js
const { fetchCoupons } = require('codice-sconto-zooplus');
fetchCoupons().then(data => console.log(data));

Come funziona lo script

1Effettua una richiesta GET alla pagina pubblica https://shopilo.it/negozi/zooplus.it con un User-Agent standard
2Analizza l'HTML con BeautifulSoup e seleziona gli elementi con selettori CSS (.coupon-item, [data-coupon])
3Estrae per ogni coupon: il codice (data-code), lo sconto, la descrizione e la data di scadenza
4Restituisce la lista come JSON in stdout, pronto per l'integrazione in qualsiasi pipeline, bot o estensione

Domande frequenti, codice sconto Zooplus

Cosa restituisce lo script fetch.py?

Un array JSON con gli oggetti dei coupon attivi: il codice sconto Zooplus, la percentuale di sconto, la descrizione, la data di scadenza e l'URL fonte da shopilo.it.

Ogni quanto posso eseguire lo script?

Consigliamo al massimo un'esecuzione ogni 6 ore. I dati su shopilo.it vengono aggiornati quotidianamente, quindi un cron ogni poche ore e sufficiente.

Funziona anche come modulo npm?

Si. npm install codice-sconto-zooplus installa la versione JavaScript che utilizza fetch() nativo da Node 18+ ed esporta la stessa struttura JSON.

Dove trovo tutti i coupon Zooplus attivi?

La lista completa e aggiornata quotidianamente su shopilo.it/negozi/zooplus.it, la fonte dati di questo script.

Posso combinare piu codici sconto Zooplus?

Di solito un solo voucher Zooplus per ordine. Le condizioni complete sono indicate su ogni coupon su shopilo.it.

Informazioni su Zooplus

Zooplus e uno dei negozi online piu popolari. Lo script recupera automaticamente i codici sconto Zooplus disponibili su shopilo.it.

Come usare un codice Zooplus

1Copia il codice dall'output dello script
2Aggiungi i prodotti desiderati al carrello Zooplus
3Al checkout, inserisci il codice nel campo "Voucher" o "Codice promozionale"
4Premi "Applica", lo sconto viene applicato automaticamente
